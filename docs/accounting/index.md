# Accounting Module

The Accounting module is the financial backbone of QAL ERP. It provides tools to manage your General Ledger, Chart of Accounts, Vouchers, and Financial Year settings.

## Module Overview

| Sub-Module | Purpose |
|------------|---------|
| [General Ledger](general-ledger) | View and manage all GL transactions |
| [Chart of Accounts](chart-of-accounts) | Manage individual GL accounts |
| [Chart of Accounts Type](chart-of-accounts-type) | Define account categories (Asset, Liability, etc.) |
| [Voucher Type](voucher-type) | Define transaction types (Payment, Receipt, etc.) |
| [Financial Year](financial-year) | Manage accounting periods |

---

## Navigation

Access the Accounting module from the left sidebar:

**Sidebar → Accounts**

All sub-modules are listed under the **Accounts** section.

---

{: .note }
Only users with the **Administrators** role or equivalent accounting permissions can access setup modules such as Chart of Accounts Type and Voucher Type.
